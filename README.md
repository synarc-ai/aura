## 🚀 Getting Started

### For Engineers
```bash
# Clone repository
git clone https://github.com/synarc-ai/aura
cd aura

# Install dependencies
npm install

# Run the website locally
cd site && npm install && npm run dev

# Run tests
npm test

# Build WebAssembly module
npm run build:wasm
```
